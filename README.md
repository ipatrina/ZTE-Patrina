# ZTE Patrina

中兴ONT配置文件实用工具

![ZTE Patrina preview](https://thumbs2.imgbox.com/25/2b/a0YJt1e4_t.png)


# 软件功能

- 支持Deflate解压缩中兴(ZTE)光终端XML、CFG格式配置文件。

- 支持使用默认AES密钥解密中兴(ZTE)光终端XML、CFG格式配置文件。

- 支持使用Indiv密钥(一机一密)解密中兴(ZTE)光终端XML、CFG格式配置文件。


# 系统要求

适用于Windows 7及以上操作系统。
